Tumbach is still in the closed beta!
At this stage Tumbach is a MVP: main features are done and working properly, and the features are in development. We are testing the engine among a limited number of people in order to identify bugs early on. The number of testers will increase according to engine's readiness.

You can send an email from any address to sup@tumba.ch to sign up for testing on a first-come, first-served basis. When testing new features, you will receive an invitation and a link in return.
You will be the first to know if open beta testing opportunities become available.
